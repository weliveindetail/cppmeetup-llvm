What is a...

JIT Compiler?

A traditional compiler...

  • transforms high-level source code into low-level machine code
  • runs prior to execution as a static compiler

A JIT Compiler...

  • transforms high-level source code into low-level machine code
  • runs during the execution of a program as a runtime compiler
  • produces executable code "Just-in-Time"
